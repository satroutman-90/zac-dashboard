# GitHub Pages Setup — One Time

Follow these steps once. After that, daily updates take ~30 seconds.

---

## Step 1: Create a GitHub repo

1. Go to [github.com](https://github.com) and sign in
2. Click the **+** icon (top right) → **New repository**
3. Name it: `zac-fitness` (or anything you like)
4. Set to **Public** ✅ (required for free GitHub Pages)
5. Click **Create repository**

---

## Step 2: Upload the dashboard file

1. In your new repo, click **Add file** → **Upload files**
2. Drag and drop `index.html` from your `Zac Fitness` folder
3. Scroll down, click **Commit changes**

---

## Step 3: Enable GitHub Pages

1. In your repo, click **Settings** (top tab)
2. In the left sidebar, click **Pages**
3. Under "Branch", select `main` → folder `/ (root)`
4. Click **Save**
5. Wait ~1 minute, then your dashboard will be live at:
   `https://YOUR-GITHUB-USERNAME.github.io/zac-fitness/`

Share that URL with Zac — it works on any phone, tablet, or computer.

---

## Daily Update (30 seconds)

1. Tell Claude Zac's numbers for the day
2. Claude updates `index.html` in your Zac Fitness folder
3. Go to your GitHub repo → click `index.html` → click the **pencil icon** (Edit)
   - OR: click **Add file** → **Upload files**, drag the new `index.html`, and commit
4. Done — dashboard updates instantly

---

## Tip: Bookmark the link

Add `https://YOUR-GITHUB-USERNAME.github.io/zac-fitness/` to Zac's phone home screen:
- **iPhone:** Open in Safari → Share → Add to Home Screen
- **Android:** Open in Chrome → Menu → Add to Home Screen
